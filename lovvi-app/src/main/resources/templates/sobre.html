Sobre o Lovvi App

Um projeto Spring Boot moderno e bem estruturado pronto para crescer.

🎯 Objetivo

Este projeto é o ponto de partida para a aplicação Lovvi, construído com Spring Boot 3 e Java 21, seguindo as melhores práticas de desenvolvimento moderno.

🏗️ Estrutura

  • Controller — Camada de apresentação (MVC)
  • Service — Regras de negócio
  • Repository — Acesso a dados
  • Templates — Views com Thymeleaf
